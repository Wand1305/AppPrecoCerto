# Preço Certo

**Descrição do projeto:**  
Este é um aplicativo Android desenvolvido como parte de um trabalho acadêmico. O aplicativo preço certo é um app onde usuários podem se cadastrar e após o cadastro, realizar o login para fazer a publicação de mercados e produtos desses mercados, informando o preço desses produtos.
Outros usuários podem consultar o produto desejado para saber o preço e poder fazer a melhor comparação entre mercados, sem sair de casa.
Os usuários podem a qualquer momento trocar a senha e até mesmo realizar a exclusão de sua conta, caso não deseje mais interagir com a plataforma.

## Instruções de Instalação

### Instalando o APK
1. Baixe o arquivo APK anexado ao projeto.
2. Habilite a instalação de aplicativos de fontes desconhecidas em seu dispositivo Android:
   - Vá em **Configurações > Segurança > Fontes desconhecidas** e ative a opção.
3. Abra o APK no seu dispositivo e siga as instruções para instalação.

### Instalando no Android Studio
Caso queira rodar o projeto diretamente no Android Studio, siga as instruções abaixo:
1. Clone ou baixe o código-fonte do projeto para sua máquina local.
2. Abra o Android Studio e selecione **Open an existing Android Studio project**.
3. Selecione a pasta onde o código-fonte do app foi salvo.
4. O Android Studio irá sincronizar o projeto com as dependências e gerar o ambiente necessário para rodá-lo.
5. Conecte um dispositivo Android ou use um emulador para executar o app.

## Funcionalidades Desenvolvidas

O aplicativo contém as seguintes funcionalidades principais:

- **Tela Login**: Usuário pode acessar com usuário e senha e também realizar o cadastro inicial, caso não tenha..
- **Tela Inicial**: Apresenta as informações principais do aplicativo, com acesso rápido às funcionalidades.
- **Cadastro de mercados e produtos**: Permite ao usuário adicionar um novo mercado ou um novo produto e o preço do produto, selecionando o mercado em que o produto está.
- **Edição e Exclusão**: O usuário pode trocar sua senha ou excluir sua conta.
- **Armazenamento Local**: As tarefas são armazenadas localmente no dispositivo usando um banco de dados SQLite.
- **Notificações**: O app notifica caso o usuário não tenha conta, ou se o usuário ou senha estão errados.

## Tecnologias Utilizadas
- **Android Studio**: A plataforma Android para o desenvolvimento do app.
- **SQLite**: Para armazenamento local de dados.
- **Kotlin**: Linguagem de programação principal utilizada no desenvolvimento.

## Como Testar o App
1. **Instalar o APK**: Após a instalação, abra o aplicativo e interaja com as funcionalidades de acordo com o fluxo descrito acima.
2. **Testes Manuais**: Teste as funcionalidades de adicionar, editar e excluir. Verifique também o funcionamento das notificações e do armazenamento local.
